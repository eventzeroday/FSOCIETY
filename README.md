# WEFARM - A CROP DISEASE DETECTION WEBSITE 


**WEFARM The Crop Disease Detection Website** is a web-based application that helps farmers and agricultural researchers identify crop diseases risk by answering questions with our chatbot. The system uses machine learning techniques to analyze the farmers input and predict the disease along with possible treatments.

This project aims to improve crop health, reduce losses, and promote smart agriculture.

## FEATURES:
- AI-based crop disease prediction.
- Supports multiple crop types.
- Displays disease risk and confidence level.
- Provides basic treatment or prevention tips.
- User-friendly and responsive web interface.

## TECHNOLOGIES USED:

- FRONTEND:
-- React, javascript.
- BACKEND:
-- FastAPi, PHP, Python.
- API used:
-- Openweather api, leaflet JS library, sentinel satellite imagery.
